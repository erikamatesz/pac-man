## Setup (MacOS)

Create a venv:

```sh
python3 -m venv pacmanenv
```

Activate the venv:

```sh
source pacmanenv/bin/activate
```

Install dependencies:

```sh
pip install -r requirements.txt
```

If you need to deactivate the venv:

```sh
deactivate
```
